## Icon Button

`icon` prop을 전달하면 아이콘 전용 버튼으로 렌더링됩니다. 아이콘 버튼은 `subtle`과 `ghost` appearance를 지원합니다.

```tsx
import { PlusIcon } from "@phosphor-icons/react";
  <Button icon={<PlusIcon />} appearance="ghost" size="small" />
<Button icon={<PlusIcon />} appearance="ghost" size="medium" />
<Button icon={<PlusIcon />} appearance="ghost" size="large" />
<Button icon={<PlusIcon />} appearance="subtle" size="small" />
<Button icon={<PlusIcon />} appearance="subtle" size="medium" />
```

```tsx
import { PlusIcon } from "@phosphor-icons/react";

<Button icon={<PlusIcon />} appearance="ghost" />
<Button icon={<PlusIcon />} appearance="subtle" />
```

## Loading

`loading` prop을 전달하면 스피너가 표시되고 버튼이 비활성화됩니다.

<div className="flex items-center gap-3 p-4 border rounded-lg">
  <Button loading>Default</Button>
  <Button loading appearance="subtle">Subtle</Button>
  <Button loading appearance="error">Error</Button>
</div>

```tsx
<Button loading>저장 중...</Button>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `appearance` | `"default" \| "subtle" \| "ghost" \| "error" \| "errorSubtle"` | `"default"` | 버튼 외형 |
| `size` | `"small" \| "medium" \| "large"` | `"small"` | 버튼 크기 |
| `icon` | `ReactNode` | - | 아이콘 버튼으로 렌더링 (children과 배타적) |
| `loading` | `boolean` | `false` | 로딩 상태 표시 |
| `disabled` | `boolean` | `false` | 비활성화 |
