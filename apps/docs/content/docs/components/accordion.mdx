## Usage

```tsx
import { AccordionRoot, AccordionItem } from "@/components/ui/accordion";

export function Example() {
  return (
    <AccordionRoot>
      <AccordionItem title="Section 1" value="1">
        Content for section 1
      </AccordionItem>
      <AccordionItem title="Section 2" value="2">
        Content for section 2
      </AccordionItem>
    </AccordionRoot>
  );
}
```

## Default Expanded

```tsx
<AccordionRoot defaultValue={["1"]}>
  <AccordionItem title="기본 펼쳐진 항목" value="1">
    이 항목은 처음부터 펼쳐져 있습니다.
  </AccordionItem>
</AccordionRoot>
```

## Multiple

```tsx
<AccordionRoot openMultiple>
  <AccordionItem title="첫 번째" value="1">내용 1</AccordionItem>
  <AccordionItem title="두 번째" value="2">내용 2</AccordionItem>
</AccordionRoot>
```

## Props

### AccordionRoot

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `defaultValue` | `string[]` | `[]` | 초기 펼쳐진 항목 |
| `value` | `string[]` | - | 제어 모드용 열린 항목 |
| `onValueChange` | `(value: string[]) => void` | - | 값 변경 콜백 |
| `openMultiple` | `boolean` | `false` | 여러 항목 동시 펼침 허용 |

### AccordionItem

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `title` | `ReactNode` | - | 트리거 텍스트 |
| `value` | `string` | - | 항목 식별자 |
| `children` | `ReactNode` | - | 패널 콘텐츠 |
