## Sizes

```tsx
<Avatar size="large">김</Avatar>  {/* 40px */}
<Avatar size="small">김</Avatar>  {/* 24px */}
```

## Colors

```tsx
<Avatar color="neutral">가</Avatar>
<Avatar color="orange">나</Avatar>
<Avatar color="green">다</Avatar>
<Avatar color="blue">라</Avatar>
```

## Image

```tsx
<Avatar src="/path/to/image.jpg" alt="User" />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `"large" \| "small"` | `"large"` | 아바타 크기 (40px / 24px) |
| `color` | `"neutral" \| "orange" \| "green" \| "blue"` | `"neutral"` | 배경 색상 |
| `src` | `string` | - | 이미지 URL |
| `alt` | `string` | - | 이미지 대체 텍스트 |
| `children` | `string` | - | 이니셜 텍스트 (첫 글자만 표시) |
