## Props

### Switch

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `ReactNode` | - | 레이블 |
| `checked` | `boolean` | - | 체크 상태 (제어) |
| `defaultChecked` | `boolean` | - | 초기 상태 |
| `onCheckedChange` | `(checked: boolean) => void` | - | 변경 콜백 |
| `disabled` | `boolean` | `false` | 비활성화 |
