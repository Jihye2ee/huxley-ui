## Installation

```tsx title="components/ui/slider.tsx"
// Copy from packages/ui/components/slider.tsx
```

## With Label

```tsx
<Slider label="Volume" showValue defaultValue={[50]} />
```

## Range Slider

```tsx
<Slider label="Price range" showValue defaultValue={[20, 80]} />
```

## Step

```tsx
<Slider step={10} defaultValue={[50]} />
```

## Disabled

```tsx
<Slider label="Disabled" showValue defaultValue={[50]} disabled />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `ReactNode` | - | 슬라이더 레이블 |
| `showValue` | `boolean` | `false` | 현재 값 표시 |
| `min` | `number` | `0` | 최솟값 |
| `max` | `number` | `100` | 최댓값 |
| `step` | `number` | `1` | 이동 단위 |
| `defaultValue` | `number[]` | - | 초기 값 |
| `value` | `number[]` | - | 현재 값 (제어) |
| `onValueChange` | `(value: number[]) => void` | - | 변경 콜백 |
| `disabled` | `boolean` | `false` | 비활성화 |
