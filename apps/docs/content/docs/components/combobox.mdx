## Usage

```tsx
import { Combobox } from "@/components/ui/combobox";

export function Example() {
  return (
    <Combobox
      items={["Option 1", "Option 2", "Option 3"]}
      placeholder="Search..."
    />
  );
}
```

## States

```tsx
<Combobox items={[]} placeholder="Disabled" disabled />
<Combobox items={[]} placeholder="Error" error />
```

## Props

### Combobox

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `items` | `T[]` | required | 선택 가능한 항목 배열 |
| `size` | `"md" \| "sm"` | `"md"` | 입력 크기 |
| `error` | `boolean` | `false` | 에러 상태 |
| `placeholder` | `string` | `"Placeholder"` | 플레이스홀더 |
| `disabled` | `boolean` | `false` | 비활성화 |
| `value` | `T \| null` | - | 선택된 값 (controlled) |
| `onValueChange` | `(value: T \| null) => void` | - | 값 변경 핸들러 |
| `getLabel` | `(item: T) => string` | `String` | 항목 라벨 변환 함수 |
