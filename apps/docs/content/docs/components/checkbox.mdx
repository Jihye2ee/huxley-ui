## Props

### Checkbox

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `ReactNode` | - | 레이블 텍스트 |
| `size` | `"default" \| "sm"` | `"default"` | 체크박스 크기 |
| `checked` | `boolean` | - | 체크 상태 (제어) |
| `defaultChecked` | `boolean` | - | 초기 체크 상태 |
| `indeterminate` | `boolean` | - | 불확정 상태 |
| `disabled` | `boolean` | `false` | 비활성화 |
| `onCheckedChange` | `(checked: boolean) => void` | - | 변경 콜백 |
