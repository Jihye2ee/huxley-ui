## Installation

```tsx title="components/ui/popover.tsx"
// Copy from packages/ui/components/popover.tsx
```

## With Close Button

```tsx
<PopoverContent>
  <PopoverClose />
  <PopoverTitle>Title</PopoverTitle>
  <PopoverDescription>Description</PopoverDescription>
</PopoverContent>
```

## Placement

```tsx
<PopoverContent side="top">...</PopoverContent>
<PopoverContent side="bottom">...</PopoverContent>
<PopoverContent side="left">...</PopoverContent>
<PopoverContent side="right">...</PopoverContent>
```

## Props

### PopoverContent

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `side` | `"top" \| "bottom" \| "left" \| "right"` | `"bottom"` | 표시 위치 |
| `sideOffset` | `number` | `8` | 트리거와의 간격 |
