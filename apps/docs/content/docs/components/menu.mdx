## With Groups

```tsx
<MenuContent>
  <MenuGroup>
    <MenuGroupLabel>File</MenuGroupLabel>
    <MenuItem>New</MenuItem>
    <MenuItem>Open</MenuItem>
  </MenuGroup>
  <MenuSeparator />
  <MenuGroup>
    <MenuGroupLabel>Edit</MenuGroupLabel>
    <MenuItem>Undo</MenuItem>
    <MenuItem>Redo</MenuItem>
  </MenuGroup>
</MenuContent>
```

## Danger Items

```tsx
<MenuItem danger>삭제</MenuItem>
```

## Props

### MenuItem

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `danger` | `boolean` | `false` | 위험 동작 스타일 |
| `disabled` | `boolean` | `false` | 비활성화 |
| `onSelect` | `() => void` | - | 선택 콜백 |

### MenuContent

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `side` | `"top" \| "bottom" \| "left" \| "right"` | `"bottom"` | 표시 위치 |
| `sideOffset` | `number` | `4` | 트리거와의 간격 |
| `align` | `"start" \| "center" \| "end"` | `"start"` | 정렬 |
