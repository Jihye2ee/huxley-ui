## Installation

```tsx title="components/ui/select.tsx"
// Copy from packages/ui/components/select.tsx
```

## Sizes

```tsx
<SelectTrigger size="md">Medium</SelectTrigger>
<SelectTrigger size="sm">Small</SelectTrigger>
```

## States

```tsx
<SelectTrigger disabled>Disabled</SelectTrigger>
<SelectTrigger error>Error</SelectTrigger>
```

## With Groups

```tsx
<SelectContent>
  <SelectGroup>
    <SelectGroupLabel>Fruits</SelectGroupLabel>
    <SelectItem value="apple">Apple</SelectItem>
  </SelectGroup>
  <SelectSeparator />
  <SelectGroup>
    <SelectGroupLabel>Vegetables</SelectGroupLabel>
    <SelectItem value="carrot">Carrot</SelectItem>
  </SelectGroup>
</SelectContent>
```

## Props

### SelectTrigger

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `"md" \| "sm"` | `"md"` | 트리거 크기 |
| `error` | `boolean` | `false` | 에러 상태 |

### SelectItem

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | - | 항목 값 |
| `disabled` | `boolean` | `false` | 비활성화 |
