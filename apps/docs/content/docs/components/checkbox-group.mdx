
## Multi Column

```tsx
<CheckboxGroup title="Fruits" columns={3}>
  <Checkbox label="Apple" value="apple" />
  <Checkbox label="Banana" value="banana" />
  <Checkbox label="Cherry" value="cherry" />
</CheckboxGroup>
```

## Disabled

```tsx
<CheckboxGroup title="Disabled" disabled>
  <Checkbox label="A" value="a" />
  <Checkbox label="B" value="b" />
</CheckboxGroup>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `title` | `ReactNode` | - | 그룹 제목 |
| `columns` | `number` | - | 그리드 열 수 |
| `value` | `string[]` | - | 선택된 값 (제어) |
| `defaultValue` | `string[]` | - | 초기 선택 값 |
| `onValueChange` | `(value: string[]) => void` | - | 변경 콜백 |
| `disabled` | `boolean` | `false` | 전체 비활성화 |
